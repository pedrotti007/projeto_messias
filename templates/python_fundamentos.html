Estruturas de Seleção

Conceito

A estrutura de seleção (ou estrutura condicional) é um bloco de código que escolhe executar uma ação entre duas ou mais possíveis, com base em uma condição.

Ela permite que o programa tome decisões, como:

"Se algo acontecer, faça isso. Caso contrário, faça aquilo."

Aplicação

Onde se aplica?

Verificar permissões de acesso

Mostrar conteúdos diferentes dependendo do usuário

Criar menus ou rotas diferentes em um site

Validar dados de entrada

Controlar fluxos lógicos (ex: se o arquivo existe, se a lista está vazia...)


Exemplos práticos:

Verificar idade: 


idade = 18
if idade >= 18:
    print("Você é maior de idade.")
else:
    print("Você é menor de idade.")

Menu de opções: 


opcao = input("Escolha [1] Iniciar | [2] Sair: ")
if opcao == "1":
    print("Iniciando...")
elif opcao == "2":
    print("Saindo...")
else:
    print("Opção inválida.")



Estrutura de Repeticao

Conceito

Estrutura de repetição (também chamada de laço ou loop) é um conceito da programação que permite executar um bloco de código várias vezes, de forma automática e controlada.

Aplicação

A estrutura de repetição é aplicada sempre que você precisa executar um mesmo conjunto de instruções várias vezes. Aqui estão alguns exemplos práticos e situações comuns onde ela é usada:

Percorrer listas, duplas ou dicionários Você pode usar for para acessar cada item de uma lista ou dicionário.


nomes = ["Ana", "Carlos", "Beatriz"]
for nome in nomes:
    print("Olá,", nome)

Executar ações até uma condição mudar Ideal com while, como em validação de entrada. 


senha = ""
while senha != "python123":
    senha = input("Digite a senha correta: ")

Exemplo de Código:

Repetição com contador (laço "for") Executa um número determinado de vezes. 


for i in range(5):  # repete 5 vezes (0 a 4)
    print(i)

Repetição condicional (laço "while") Repete enquanto uma condição for verdadeira. 


x = 0
while x < 5:
    print(x)
    x += 1


Vetores e Matriz

Conceito

Vetor

Um vetor é uma sequência de dados organizados em uma única linha, onde cada elemento pode ser acessado por um índice.

Aplicação:


vetor = [10, 20, 30, 40]
print(vetor[0])  # Acessa o primeiro elemento (10)

Matriz

Uma matriz é como uma tabela, com linhas e colunas. É um vetor de vetores.

Aplicação:


matriz = [
    [1, 2, 3],
    [4, 5, 6],
    [7, 8, 9]
]
print(matriz[0][1])  # Acessa o valor 2 (linha 0, coluna 1)

Funções e Procedimentos

Conceito:

Funções e procedimentos são blocos de código reutilizáveis criados para executar tarefas específicas em um programa.
Eles ajudam a organizar, reaproveitar e modularizar o código.

Aplicação de uma função

Uma função realiza uma tarefa e retorna um valor.


    def somar(a, b):
    return a + b

resultado = somar(3, 5)
print(resultado)  # Saída: 8

Aplicação de um procedimento

É uma função que apenas executa ações, sem retornar valor (o foco é o efeito, não o resultado).


def saudacao(nome):
    print(f"Olá, {nome}!")

saudacao("Lucas")


Tratamento de Exceções

Conceito:

O tratamento de exceções é um mecanismo da programação que permite lidar com erros de forma controlada, evitando que o programa pare de funcionar bruscamente.

Sem tratamento de exceções, qualquer erro (ex: divisão por zero, dado inválido, arquivo ausente) faz o programa travar. Com o tratamento, você pode:
Mostrar mensagens amigáveis.

Corrigir o problema automaticamente.

Ignorar ou registrar o erro.

Sintaxe em Python:


try:
    # código que pode gerar erro
    resultado = 10 / 0
except ZeroDivisionError:
    # o que fazer se der erro
    print("Não é possível dividir por zero.")

Exemplo com entrada de dados:


      try:
    numero = int(input("Digite um número: "))
    print("Você digitou:", numero)
except ValueError:
    print("Erro: Isso não é um número inteiro.")
